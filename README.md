# ParentTime Legal Pages (GitHub Pages)

Este pacote publica páginas legais com visual profissional e URLs estáveis:

- `/privacy-policy/`
- `/terms-and-conditions/`

## Publicação no repositório de políticas

Use o repositório: `Jefte-Felipe/politica-de-privacidade`.

1. Copie o conteúdo desta pasta `legal-pages/` para a raiz do repositório de políticas.
2. No GitHub, acesse **Settings > Pages**.
3. Em **Build and deployment**, selecione:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `(root)`
4. Salve e aguarde o deploy.
5. URLs finais esperadas:
   - `https://jefte-felipe.github.io/politica-de-privacidade/privacy-policy/`
   - `https://jefte-felipe.github.io/politica-de-privacidade/terms-and-conditions/`

## Observações jurídicas

- Revise o conteúdo com apoio jurídico antes de produção.
- Sempre mantenha a data de “Última atualização” sincronizada com cada alteração.
